1. 基本使用

设置ui-popover属性，为任意元素添加一小块浮层，用于存放非主要信息。

2. 显示位置

设置popover-placement属性，控制弹出方向。

3. 显示弹出框标题

添加元素popover-title属性使弹出框显示标题。

4. 触发事件

通过使用 popover-trigger="focus" 触发器可以在失焦时让弹出框消失。其它触发事件的值有：mouseenter, click, none

实现“失焦时消失”的效果需要一些额外的代码

为了更好的跨浏览器和跨平台效果，你必须使用 <a> 标签，不能使用 <button> 标签，并且，还必须包含 role="button" 和 tabindex 属性。

6. 指令
指令名 默认值 说明
ui-popover null 弹出框的文字内容。会自动转义其中的html，如果需要包含html，请使用ui-popover-template 或 ui-popover-html
ui-popover-template null 使用模版显示弹出框内容。注意：模版中需要有一个根元素包裹其中的内容。
ui-popover-html null 绑定弹出框的内容，支持html。注意：传入的值为$scope中的变量名，并且需要通过$sce.trustAsHtml()方法进行转换。

7. 属性
属性名 默认值 说明
popover-title null 弹出框标题
popover-placement top 弹出框的显示位置。可选值: top, bottom, left, right
popover-animation true 是否使用动画淡入(淡出)显示(隐藏)弹出框
popover-popup-delay 0 延迟显示弹出框
popover-trigger mouseenter 触发方式。可选值: mouseenter, focus, click, none
popover-append-to-body false 是否将弹出框的代码插入<body>中。默认插入到当前元素的父级元素中。用于处理弹出框所在容器overflow:hidden情况。
popover-is-open false 是否默认打开弹出框
