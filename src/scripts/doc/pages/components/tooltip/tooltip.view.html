1. 基本使用

将ui-tooltip属性添加到<button>, <a>, <span> 等，为元素添加提示效果。

2. 显示位置

添加ui-tooltip-placement属性，设置提示框显示位置，可选值为left, right, top, bottom

3. 触发事件
显示触发事件 隐藏触发事件
mouseenter mouseleave
click click
focus blur
none ''

3. 属性
属性名 默认值 说明
ui-tooltip null 提示框的文字内容。会自动转义其中的html，如果需要包含html，请使用tooltip-template 或 tooltip-html
ui-tooltip-template null 使用模版显示提示框内容。注意：模版中需要有一个根元素包裹其中的内容。
ui-tooltip-html null 绑定提示框的内容，支持html。注意：传入的值为$scope中的变量名，并且需要通过$sce.trustAsHtml()方法进行转换。
ui-tooltip-placement top 提示框的显示位置。可选值: top, bottom, left, right
ui-tooltip-animation true 是否使用动画淡入(淡出)显示(隐藏)提示框
ui-tooltip-popup-delay 0 延迟显示提示框
ui-tooltip-trigger mouseenter 触发方式。可选值: mouseenter, focus, click, none
ui-tooltip-enable null 是否启用提示组件
ui-tooltip-append-to-body false 是否将提示框的代码插入<body>中。默认插入到当前元素的父级元素中。用于处理提示框所在容器overflow:hidden情况。
ui-tooltip-is-open false 是否默认打开提示框
ui-tooltip-class null 使用自定义class
