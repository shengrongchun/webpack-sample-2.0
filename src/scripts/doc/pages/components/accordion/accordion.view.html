4. <ui-accordion-group> 属性
属性名 默认值 说明
close-others true 当展开一个栏目是否收起其它已展开的栏目
icon true 是否显示图标
icon-position left 图标显示位置，可选值: left, right
template-url 默认模版URL 重写手风琴组件Html模版URL

5. <ui-accordion> 属性
属性名 默认值 说明
heading 或 <ui-accordion-heading> null 使用heading属性设置选项卡标题，或使用html标签<ui-accordion-heading>进行设置。
is-open false 默认展开该栏目
template-url 默认模版URL 重写手风琴栏目Html模版URL
