弹出对话框

  1. 首页
  2. Web组件
  3. 弹出对话框

1. 基本使用
该组件继承自uiModal

调用uiAlert服务，弹出对话框。

uiAlert提供以下几种方法：

  • 确认框: uiAlert.confirm(title)
  • 通知框: uiAlert.info(title)
  • 成功框: uiAlert.success(title)
  • 警告框: uiAlert.warning(title)
  • 错误框: uiAlert.error(title)
  • 输入框: uiAlert.prompt(title)

以上方法返回值均为promise类型，可以通过.then()方法回调成功事件，通过.catch()方法调用失败事件。

注意：uiAlert为Provider服务，使用时需要依赖注入。

2. 参数
参数名 类型 默认值 说明
title string 标题
settings object confirmText: 确认按钮的文字，如果传false或空则隐藏该按钮,
cancelText: 取消按钮的文字，如果传false或空则隐藏该按钮,
backdrop: 是否允许点击遮罩层关闭弹窗,
backdropClass: 为遮罩层添加class,
windowClass: 为弹出框添加class

3. 方法
方法名 说明
uiAlert.confirm() 确认框
uiAlert.success() 成功框
uiAlert.warning() 警告框
uiAlert.error() 错误框
uiAlert.prompt() 输入框，输入框的结果通过.then()回调事件中的参数返回。
