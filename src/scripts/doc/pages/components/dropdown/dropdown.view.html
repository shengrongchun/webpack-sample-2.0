    4. 下拉框中显示其他组件

    任何组件或元素上都可使用ui-dropdown-menu，用来创建更为复杂的下拉框。如下拉框中显示表单或表格

    在弹出表单后，需要点击输入框及按钮，此场景下需要阻止下拉框的自动关闭，可在ui-dropdown指令上添加属性auto-close="outsideClick"
    ui-dropdown-menu指令包含默认的min-width:150px，因此在显示表格、表单等宽度自适应的组件时需要设置固定宽度。

    5. ui-dropdown 属性
    属性名 默认值 说明
    is-open false 是否默认显示下拉框。
    auto-close always always: 点击任何元素都会触发关闭下拉框；outsideClick: 点击下拉框以外的元素触发关闭下拉框；disabled: 禁用自动关闭，但是可以通过is-open属性控制是否显示下拉框。

    6. ui-dropdown 指令
    指令名 说明
    dropdown-append-to-body 将下拉框html插入到body中。用于处理下拉框所在容器overflow:hidden情况。
    ui-keyboard-nav 使下拉框选项支持使用键盘上下键选择。
