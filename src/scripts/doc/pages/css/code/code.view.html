5. JavaScript代码块

6. Java代码块

9. 设置代码显示风格

使用指令属性theme设置代码显示风格。

可选风格

  • dark: 深色风格(默认)
  • light: 浅色风格
                    public class HelloWorld {
                        public static void main(String args[]) {
                            System.out.println("Hello World!");
                        }
                    }

10. ui-syntaxHighlighter 属性
属性名 默认值 说明
line-numbers true 是否显示行数
theme dark 选择代码显示颜色风格，可选值dark, light
